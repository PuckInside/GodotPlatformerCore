# GodotPlatformerCore

Простой проект на движке Godot, где реализовано базовое движение персонажа игрока для 2D платформера.
Вы свободно можете использовать примеры кода или сам проект как базу для своей игры.

# Состояние проекта

## Разрешение экрана

В проекте настроена работа с разрешением окна для пиксельной графики конкретно под 320х240p. На самом деле, реальное разрешение экрана 320х240 умноженный на 4, поэтому камера использует зум х4. 
Это сделано, чтобы минимизировать дрожание камеры, при плавном следованием за игроком. Масштабирование настроен так, чтобы не было чёрных полос на экране.

## Игрок

Персонаж игрока может передвигаться, прыгать и совершатьт рывок. Применяет движение к игроку машина состояний MovementFSM.
У персонажа есть время кайота и буфер ввода для прыжка. Игрок может проходить через one way collision, просто включая и выключая коллизию с нимю.
Для one way collision я выделил 5 слои коллизий.
