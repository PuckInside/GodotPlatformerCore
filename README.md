# GodotPlatformerCore

Простой проект на движке Godot, где реализовано базовое движение персонажа игрока для 2D платформера.
Вы свободно можете использовать примеры кода или сам проект как базу для своей игры.

# Состояние проекта

## Разрешение экрана

В проекте настроена работа с разрешением окна для 2D-игр. Поддерживаются пиксельные разрешения и нестандартные соотношения экрана, например 16:10.
Окно изначально запускается в 2K-разрешении, что позволяет избежать графических артефактов, а также лучше уменьшает изображение для пиксельной графики по сравнению с FullHD.
Такое разрешение в теории может замедлить игру, но это компенсируется удобством разработки.
