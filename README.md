# GodotPlatformerCore

Простой проект на движке Godot, где реализованно базовое движение персонажа игрока для 2D платформера.
Вы свободно использовать примеры кода или же сам проект, как базу для своей игры. 

# Состояние проекта

## Разрешение экрана

В проекте настроено разрешение окна для 2D игр, также поддерживает пиксельные разрешения и нестандартные соотношения экрана, например 16:10.
Окно изначально работает в 2К, такое решение обуславливаеться тем, что позволяет избежать графические артефакты, а также лучше уменьшаться для пиксельной графики, чем FullHD.
